# 企微自动结束会话工具 - 迁移指南

## 需要复制的文件

将整个 `wecom_auto_end/` 文件夹复制到工作电脑，包含：

```
wecom_auto_end/
├── main.py              # 主程序
├── config.yaml          # 配置文件（需要修改）
├── requirements.txt     # Python依赖
├── wecom_api.py        # 企微API封装
├── session_manager.py  # 会话管理
├── scheduler.py        # 定时任务
├── logger.py           # 日志模块
└── utils.py            # 工具函数
```

## 工作电脑环境准备

### 1. 安装 Python（如果没有）
```bash
# macOS
brew install python@3.11

# 或从官网下载：https://www.python.org/downloads/
```

### 2. 安装依赖
```bash
cd wecom_auto_end
pip3 install -r requirements.txt
```

### 3. 修改配置文件
编辑 `config.yaml`，填入你的企微信息：

```yaml
wecom:
  corp_id: "你的企业ID"
  secret: "你的应用Secret"
  
auto_end:
  enabled: true
  schedule: "0 18 * * 1-5"  # 每周一到五 18:00
  idle_hours: 24
  
logging:
  level: "INFO"
  file: "logs/wecom_auto.log"
```

## 权限设置

### macOS 需要的权限：
1. **辅助功能权限**（如果需要自动化操作）
   - 系统设置 → 隐私与安全性 → 辅助功能
   - 添加 Terminal 或你使用的终端应用

2. **网络权限**（自动获取，无需手动设置）

## 启动方式

### 方式一：手动启动
```bash
cd wecom_auto_end
python3 main.py
```

### 方式二：后台运行
```bash
cd wecom_auto_end
nohup python3 main.py > logs/output.log 2>&1 &
```

### 方式三：开机自启动（推荐）
创建 LaunchAgent：

```bash
# 创建配置文件
cat > ~/Library/LaunchAgents/com.wecom.autoend.plist << 'EOF'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.wecom.autoend</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/python3</string>
        <string>/Users/你的用户名/wecom_auto_end/main.py</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>KeepAlive</key>
    <true/>
    <key>StandardOutPath</key>
    <string>/Users/你的用户名/wecom_auto_end/logs/stdout.log</string>
    <key>StandardErrorPath</key>
    <string>/Users/你的用户名/wecom_auto_end/logs/stderr.log</string>
</dict>
</plist>
EOF

# 加载服务
launchctl load ~/Library/LaunchAgents/com.wecom.autoend.plist

# 启动服务
launchctl start com.wecom.autoend
```

## 验证运行

```bash
# 查看日志
tail -f wecom_auto_end/logs/wecom_auto.log

# 查看进程
ps aux | grep main.py
```

## 常见问题

**Q: 提示缺少模块？**
```bash
pip3 install --upgrade -r requirements.txt
```

**Q: 权限被拒绝？**
- 检查 config.yaml 中的企微凭证是否正确
- 确认应用有"客户联系"权限

**Q: 定时任务不执行？**
- 检查 cron 表达式格式
- 查看日志文件确认调度器是否启动

## 停止服务

```bash
# 如果用 launchctl
launchctl stop com.wecom.autoend
launchctl unload ~/Library/LaunchAgents/com.wecom.autoend.plist

# 如果是后台进程
pkill -f "python3 main.py"
```
