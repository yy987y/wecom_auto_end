## 关闭流程详细

```
开始关闭流程
  │
  ▼
Whistle HTTP 拉取最近请求
  │
  ├─ 请求数 > 0？
  │   │
  │   ▼ 否
  │  ❌ 失败：无请求
  │
  ▼ 是
解析请求数据
  │
  ├─ 提取 token
  ├─ 提取 cookie
  └─ 提取 sessionId
  │
  ▼
验证数据完整性
  │
  ├─ token 存在？
  ├─ cookie 存在？
  └─ sessionId 存在？
  │
  ▼ 全部存在
构造 API 请求
  │
  ├─ URL: /chat/api/session/closeWXCSSession
  ├─ Method: POST
  ├─ Headers: Cookie
  └─ Body: {sessionId, groupManageId}
  │
  ▼
发送 HTTP 请求
  │
  ├─ 状态码 200？
  │   │
  │   ▼ 否
  │  ❌ 失败：API 错误
  │
  ▼ 是
解析响应
  │
  ├─ code == 200？
  │   │
  │   ▼ 否
  │  ❌ 失败：业务错误
  │
  ▼ 是
✅ 成功关闭
  │
  ▼
记录日志
  │
  ▼
返回主流程
```

## 本地判断流程

```
输入：群名 + 消息列表
  │
  ▼
初始化评分 = 0
  │
  ▼
遍历每条消息
  │
  ├─ 包含"好的/OK/收到"？ → 评分 +3
  ├─ 包含"谢谢/感谢"？ → 评分 +2
  ├─ 包含"稍后/回复您"？ → 评分 +2
  ├─ 包含"已修复/已处理"？ → 评分 +2
  └─ 包含"还有/另外"？ → 评分 -5
  │
  ▼
计算最终评分
  │
  ├─ 评分 ≥ 4？ → strong_end_candidate
  ├─ 评分 < 0？ → not_end
  └─ 其他 → uncertain
  │
  ▼
输出三态结果
```

```
  ▼

  │
  ▼

```

## UI 自动化流程

```
开始 UI 自动化
  │
  ▼

  │
  ├─ 执行 wecom_click_sidebar_candidate.swift

  └─ 模拟点击
  │
  ▼
等待 1 秒
  │
  ▼
切换到网易智企
  │
  ├─ 执行 wecom_click_netease.swift
  ├─ 查找"网易智企"标签
  └─ 模拟点击
  │
  ▼
等待 2 秒
  │
  ▼

  │
  ├─ 查找"重新登录"按钮
  │   │
  │   ▼ 存在

  │
  ▼

  │
  ├─ Whistle 有 qiyukf 请求？

  │

```

## 错误处理流程

```
捕获异常
  │
  ├─ Whistle 连接失败
  │   └─> 检查 Whistle 是否运行
  │       └─> 检查系统代理设置
  │
  ├─ Accessibility 权限不足
  │   └─> 提示授予权限
  │       └─> 重启程序
  │
  ├─ Cookie 过期
  │   └─> 重新登录网易智企
  │       └─> 重新提取 Cookie
  │
  ├─ API 调用失败
  │   └─> 检查网络连接
  │       └─> 检查 sessionId 有效性
  │
  └─ 未知错误
      └─> 记录完整堆栈
          └─> 继续监听（不中断）
```
