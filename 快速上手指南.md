# 企微自动结束会话 - 快速上手指南

## 📌 这是什么？

一个自动化工具，帮你智能判断并关闭企微七鱼客服会话。

**核心功能：**
- 🤖 自动监听企微会话切换
- 🧠 智能判断是否需要关闭（本地规则 + AI）
- ⚡ 自动调用 API 关闭会话
- ⏰ 20分钟无回复自动关闭

## 🚀 5分钟快速开始

### 1. 下载项目

```bash
git clone https://github.com/yy987y/wecom_auto_end.git
cd wecom_auto_end
```

### 2. 一键启动

```bash
./start.sh
```

首次运行会自动：
- 安装依赖
- 启动 Whistle 代理
- 安装证书（需要输入密码）

### 3. 授予权限

系统会提示授予"辅助功能"权限：
- 打开"系统设置 → 隐私与安全性 → 辅助功能"
- 勾选 Terminal.app 或 iTerm.app

### 4. 配置 Brainmaker（AI 判断）

程序会自动从浏览器提取 Brainmaker Cookie，但首次需要：

1. 浏览器访问 https://brainmaker.ai
2. 登录你的账号
3. 保持浏览器登录状态

程序会自动读取 Cookie，无需手动配置。

### 5. 开始使用

程序启动后会自动监听企微会话，无需手动操作。

## 🎯 工作原理

```
切换企微会话
    ↓
自动读取聊天内容
    ↓
智能判断是否需要关闭
    ├─ 客户说"好的"、"谢谢" → 关闭
    ├─ 我方说"有结论随时同步" → 关闭
    ├─ 20分钟无回复 → 关闭
    └─ 不确定 → AI 判断
    ↓
自动调用 API 关闭会话
```

## ✅ 什么情况会自动关闭？

1. **客户确认**：好的、OK、收到
2. **客户感谢**：谢谢、感谢
3. **我方承诺**：有结论随时同步、稍后联系
4. **问题已解决**：已修复、已处理
5. **超时无回复**：我方发消息后 20 分钟无客户回复

## ❌ 什么情况不会关闭？

1. **客户有新问题**：还有个问题、另外问一下
2. **对话不明确**：交给 AI 判断
3. **消息太少**：少于 2 条有效消息

## ⚙️ 配置调整

### 修改超时时间（默认 20 分钟）

编辑 `full_auto.py`，搜索 `>= 20`，改为你想要的分钟数。

### 添加自定义关键词

编辑 `wecom_judge.py`：
- `CONFIRM_WORDS`：客户确认词
- `PROMISE_WORDS`：我方承诺词
- `SOLVED_WORDS`：问题已解决词

## 🐛 常见问题

### 1. 提示"需要 Accessibility 权限"
→ 系统设置 → 隐私与安全性 → 辅助功能 → 勾选 Terminal

### 2. Whistle 无法捕获请求
→ 运行 `./repair_whistle_env.sh` 修复代理设置

### 3. 关闭失败，提示"session is timeout"
→ 等待下次会话切换，程序会自动刷新数据

### 4. AI 判断失败，提示"未获取到 Brainmaker cookies"
→ 浏览器访问 https://brainmaker.ai 并登录
→ 保持浏览器登录状态
→ 重启程序

### 5. 启动很慢
→ 首次启动需要安装依赖，二次启动会快很多

## 📈 效果预期

- **准确率**：本地规则 + AI 兜底，准确率 > 90%
- **响应速度**：会话切换后 2-5 秒内完成判断
- **AI 调用**：每天约 10-30 次（大部分由本地规则处理）
