## 组件说明

### 监控层
- **wecom_monitor.py**: 使用 macOS Accessibility API 读取企微窗口内容
- **AppleScript**: 监听会话切换事件

### 执行层
- **wecom_executor.py**: 协调 Swift 脚本完成 UI 自动化
- **Swift 脚本**: 通过 Accessibility API 模拟点击

### 判断层
- **wecom_judge.py**: 本地关键词评分，快速判断
- **wecom_agent.py**: AI 兜底，处理不确定情况

### 数据层
- **Whistle**: HTTP 代理，捕获七鱼 API 请求
- **cookie_manager.py**: 管理 Brainmaker Cookie

### API 层
- **七鱼 API**: 关闭会话接口
- **Brainmaker API**: AI 判断接口

## 数据流

```
企微会话切换
  ↓
[监控层] 读取群名和消息
  ↓
[判断层] 本地评分 → 三态输出
  ↓
uncertain? → [判断层] AI 兜底
  ↓
需要关闭? → [执行层] UI 自动化
  ↓
[数据层] Whistle 捕获请求
  ↓
[执行层] 调用关闭 API
  ↓
[日志层] 记录结果
```

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 监控 | macOS Accessibility API | 读取窗口内容 |
| 监控 | AppleScript | 监听会话切换 |
| 执行 | Swift + Accessibility | UI 自动化 |
| 判断 | Python 正则 | 关键词匹配 |
| 判断 | Brainmaker API | AI 兜底 |
| 数据 | Whistle (Node.js) | HTTP 代理 |
| API | requests (Python) | HTTP 客户端 |
| 日志 | logging (Python) | 日志记录 |
